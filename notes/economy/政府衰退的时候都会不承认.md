# 政府衰退的时候都会不承认

经济衰退时，全球各国政府的本能反应几乎一致：先是否认，再美化或改写叙事，最后才在迫不得已时承认问题并寻找“背锅侠”。

## 为什么政府一定要嘴硬
- **信心管理（自我实现的预言）**：公开承认衰退会加剧恐慌、触发股市暴跌和挤兑，因此即便形势严峻也要对外宣称“稳中向好”。
- **政治甩锅**：承认衰退意味着执政失败。选举制国家担心失去选票，非选举制国家则担心执政合法性受质疑。

## 否认三部曲（常见 SOP）
1. **死不承认——“基础稳固论”**：强调“基本面没变”“韧性强”，即使数据已显疲弱。2007 年美国次贷危机前夕、苏联解体前的报喜不报忧都属此类。
2. **发明新词——“重新定义论”**：当负增长难以遮掩，就换个名字。例如 2022 年美国将连续两季 GDP 负增长称为“向更稳定增长的过渡”，中国常用“结构性优化”“高质量发展转型期”。
3. **承认但甩锅——“外部环境论”**：实在挡不住时，才承认经济有问题，并归因于疫情、战争、上一届政府或其他外部冲击。

## 听懂官方“黑话”
| 官方辞令 | 实际含义 |
| :--- | :--- |
| 经济面临下行压力 | 经济已经在衰退。 |
| 待就业 / 灵活就业 | 失业率很高，岗位不足。 |
| 负增长 / 收缩 | 就是衰退，只是避开“衰”字。 |
| 流动性合理充裕 | 贷款需求疲弱，资金在银行空转。 |
| 软着陆 | 试图避免坠机，但风险仍大。 |
| 暂时性通胀 | 物价要涨，政策承认失误。 |

## 何时会主动承认衰退
唯一例外是政府想推出大规模刺激时，需要强调形势严峻以争取预算、压低民众对通胀的抵触。例如 2008 年雷曼倒闭后，美国迅速承认危机以推动 7000 亿美元救市计划。

## 观察行动比听口号更重要
- 央行频繁降息或扩表、政府发行超长期国债、密集出台促消费和房地产托底政策，往往意味着决策层已把衰退当作既定事实。
- “嘴上说是小感冒，身体却在进 ICU”：话术再柔和，也掩盖不了政策上的紧急动作。
